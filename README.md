📊 Netflix Data Analysis Project

📌 About Project
This project analyzes Netflix Movies and TV Shows dataset using Python.

 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

📊 Exploratory Data Analysis (EDA)

The following visualizations were used:

- countplot→ to compare Movies vs TV Shows
- pie chart → to show percentage distribution
- bar plot → to analyze top countries
- bar plot→ to analyze top genres

🎯 Key Insights
- Movies are more than TV Shows
- USA is the top content-producing country
- Drama and International Movies are most common genres

🚀 Conclusion
This project helps understand Netflix content distribution using data visualization techniques.
